# Modified N-Queen

## Project Description

This project implements a modified version of the N-Queen problem using multiple search algorithms, including:

- Breadth-First Search (BFS)
- Uniform Cost Search (UCS)
- A* Search
- Basic Heuristic

In this modified version, all queens are initially placed in the starting row.  
The goal is to move the queens to a state where no two queens attack each other.

The project demonstrates different search strategies and heuristic techniques to compare performance.

---

## How to Compile and Run

To compile the project, build the source files to generate `main.exe`.

To run the program:

1. Open **Command Prompt (cmd)**
2. Navigate to the project folder
3. Run: main.exe

---

## Project Structure

- `main.cpp` – Entry point of the program
- `State.h` – State and Action definitions
- `Utilities.h` – Helper functions
- `BFS.h` – Breadth-First Search
- `UCS.h` – Uniform Cost Search
- `AStar.h` – A* Search
- `Heuristics.h` – Heuristic function
